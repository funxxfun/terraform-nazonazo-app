# 🏗️ Terraform クイズ

3分間でTerraformの知識を試す対話型クイズアプリです。

## 🎮 デモ

コマンド・用語・選択問題をランダムに出題。タイムアップまでに何問正解できるか挑戦しよう！

---

## ✨ 機能

- **3分間タイムアタック** — カウントダウンタイマーで緊張感のあるクイズ体験
- **2段階の難易度** — 初級・中級から選択可能
- **3種類の問題形式**
  - ⌨️ コマンド問題（`terraform ___`の穴埋め）
  - 📖 用語問題（Terraformの概念・用語）
  - 🔘 選択問題（A/B/Cから選ぶ）
- **柔軟な採点** — 部分一致・大文字小文字・全角半角・カタカナ/ひらがなを自動で正規化
- **ヒント表示** — 正解・不正解どちらでも解説を表示
- **キーボード操作対応** — Enterキーで送信・次の問題へ

---

|--------|---------|------|------|------|

## 🚀 セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/your-username/terraform-quiz.git
cd terraform-quiz

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## 🤝 コントリビューション

問題の追加・修正は `src/App.tsx` 内の `QUIZZES` 配列を編集してください。

```ts
{ level:"初級", type:"コマンド", q:"問題文", a:"答え", hint:"解説" }
```
